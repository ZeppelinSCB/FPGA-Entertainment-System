# FPGA Entertainment System
## Introduction
This project aim to create a Snake Game on Cyclone IV FPGA Chips. The game should have following feature:
+ User Interface
  + Main Menu
+ In Game Experience
  + Score Display
+ Visual
  + 8 bit color image
